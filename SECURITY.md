# Security Policy

## Supported versions

Only the `main` branch is supported. Security fixes will be applied there
and deployed to <https://技術者倫理.com>.

## Reporting a vulnerability

**Please do not open a public issue for security problems.**

Instead, report privately via either of the following:

- GitHub Security Advisory: <https://github.com/EdamAme-x/rinri/security/advisories/new>
- A direct message to the maintainer (EdamAme-x)

You will receive an initial response within **72 hours**.
After we confirm the impact, we will share a remediation plan and timeline.

## Disclosure policy

We follow coordinated disclosure. Once a fix is released, we will publish a
GitHub Security Advisory describing the issue. Reporters will be credited
unless they prefer to remain anonymous.
